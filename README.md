# Robo Cafe - internal app

Google Apps Script web app for Certain Ventures' robo-kiosk coffee operations (inventory, kiosk stock reports, servicing, schedule, reimbursements, backups).

- `Code.gs` - Apps Script backend
- `Index.html` - single-file frontend served via HtmlService

## Deploy
Paste both files into the Apps Script editor, then Deploy -> Manage deployments -> New version.
